# Electric Bill Estimator

A lightweight, local utility script designed to estimate monthly electricity costs based on current regional tariffs and power consumption metrics. 

## Features
* **Regional Tariff Calculation:** Configured for local electricity board slabs (e.g., MPPKVVCL Indore zone tariffs).
* **Energy Consumption Calculator:** Input your average daily kilowatt-hour (kWh) usage or individual appliance ratings to estimate monthly power draw.
* **Fixed & Variable Cost Breakdown:** Computes energy charges alongside mandatory fixed service charges, duties, and taxes for an accurate final estimate.
